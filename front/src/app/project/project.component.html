Stage Moove 2023
Application web Moove
Application web de gestion et de réservation de vols d’affaires. Développement complet de nouvelles features, débogage suivant les tests utilisateurs. Personnalisation selon les demandes client. Synchronisation des données des opérateurs avec leurs Operating Softwares. Responsive design (utilisation de Tailwind CSS, version mobile build avec Capacitor). Validation des schémas de données front avec Yup et Formik. Validation backend avec Zod. Installation et implémentation de tests end-to-end avec Cypress. Utilisation de l’ORM Prisma. Mailing automatiques et dynamiques via SendGrid, écriture de templates HTML en Nunjucks.
